Input

The first line of input consists of a single integer N (1≤N≤200000). The next line consists of N space-separated integers a1,a2,…,aN.

凝视默尔索过久，默尔索会回以凝视

Posted on 2016-05-02 Edited on 2021-12-04

    在我们在阅读加缪的《局外人》时，不可避免会做的一件事就是评判默尔索。我读的时候也不断思考：默尔索是个好人么？默尔索是否该被判处死刑？默尔索的行为合理么？默尔索这么说这么想反映了他的一种怎样的性格？

    但是，昨天我突然想到，这种思维的方式本身就是一种错误。

    我感觉，默尔索是这部小说的基石，我们需要通过默尔索，来评判司法的制度，甚至是评判社会评判人性。但是唯独不能评判的是默尔索，一旦对他有任何观点，我们也就成了具有人性的和具有社会的荒诞的”群众“或是”司法制度“类似的东西。荒诞的主体，正如加缪想要揭示的那样，不是一个坏的审判，而是审判本身。无论我们如何客观如何科学地审视默尔索的事件，一种强烈的荒诞的感觉都会出现。只有从他的角度，我们才能更好的感到世界巨大的荒诞，进而反思人在其中应该如何生活。

关于几种品质

Posted on 2015-09-28 Edited on 2021-12-04

  家父是个基督徒。作为一个年轻而虚无的不可知论者，昨天我冒着与教徒圣战的风险和他聊了聊诚实的问题，结果一聊就偏了，最后我们讨论了品质这个词语的意思，然后什么结论也没有得出来。他一直在扯基督教，我一直在扯语言，聊得十分爽。于是我写这篇文章以记之。目的是重新整理一下思路 ，兼以展示我们平时聊天的水平（捂脸）。

  首先我们观察到这样一个现象：在《圣经》中，提到的要求大致可以分为两种，即“如何对待他人”和“如何对待神”。前面一种，比方说要爱你的邻人，要给别人另一边脸去打等等许多，后一种，说的是要追随上帝，无条件的相信他，不要追求钱财而去侍奉神，等等。前者大致形成了一种尘世间的道德体系，他们可以归纳成一些美好的品质。但很奇怪的一点是，整本圣经提到了多处大意是“你要对神诚实”的句子，但没有一处要求我们对别人也要诚实。诚实并没有被归为某种道德品质，而是被归为对神的态度。于是问题来了：对于圣经的作者来说，诚实到底意味着什么？
