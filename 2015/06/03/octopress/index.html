  • 搭建本站的工具链是这样的:Mac OS X + Github PagesRuby + Gem + Bundle + Git + Homebrew + iTerm2 + Rake + Github(app) + DNSpod + 万网 + Macdown + vim + Markdown + HTML + CSS + Xcode.搭这样一个网站真的会让人盲目自信.

  • 哦博客搭好啦,如果你想要发文章呢,你就先输入cd octopress,然后输入rake new_post[“Post Title”],然后cd source/_posts/接着输入vim yyyy-mm-dd-Post-Title.markdown,然后就可以用VIM编辑器编辑markdown文件啦!写好之后只要输入sudo rake gen_deploy最后再同步一下就结束啦!记住修改元标签啊~权限不够只要加 sudo就可以了哦!是不是很简单啊?
