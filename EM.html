高斯混合概型参数估计的 EM 算法¶

2030773 李昊洋¶

实验要求¶

  1. 编码实现高斯混合概型估计的 EM 算法
  2. 利用 Sklearn 中的 make_blobs 方法生成高斯混合样本集，用于参数估计。
  3. 调用 1）中自编的参数估计函数，对 2）中生成的混合样本集进行参数估计。

算法流程¶

输入：混合样本集 $\mathcal{D}=\left\{Y_{1}, Y_{2}, \cdots, Y_{n}\right\},$ 类别数c，间值 $\varepsilon$

输出 : 各类的分布参数 $\mu_{j} 、 \Sigma_{j}$ 和 $P\left(\omega_{j}\right), \quad j=1,2$

过程:

(1)选择初始估计值： $\theta=\theta^{(0)}=\left(P^{(0)}, \mu_{1}^{(0)}, \mu_{2}^{(0)}, \Sigma_{1}^{(0)}, \Sigma_{2}^{(0)}\right), P=\left(P_{1}^{(0)}, P_{2}^{(0)}\right)^{T} ;$ 令 $\mathrm{t}=0$ 。

(2)重复执行下列步骤：

  1. 计算 $T_{j, i}^{(t)}:=p\left(\mathbf{z}_{i}=\omega_{j} \mid \boldsymbol{Y}_{i}, \theta^{(t)}\right)=\frac{P_{j}^{(t)} f\left(\boldsymbol{Y}_{i} \mid \mathbf{\mu}_{j}^{(t)}, \Sigma_{j}^{(t)}\right)}{P_{1}^{(t)} f\left(Y_{i} \mid \mathbf{\mu}_{1}^{(t)}, \Sigma_{1}^{(t)}\right)+P_{2}^{t} f\left(Y_{i} \mid \mathbf{\mu}_{2}^{(t)}, \Sigma_{2}^{(t)}\right)}, \quad i=1,2, \cdots, n$
  2. 计算 $P_{j}^{(t+1)}=\frac{1}{n} \sum_{i=1}^{n} T_{j, i}^{(t)}, j=1,2$
  3. 计算 $\mu_{j}^{(t+1)}=\frac{\sum_{i=1}^{n} T_{j, i}^{(t)} Y_{i}}{\sum_{i=1}^{n} T_{j, i}^{(t)}}, \quad \Sigma_{j}^{(t+1)}=\frac{\sum_{i=1}^{n} T_{j, i}^{(t)}\left(Y_{i}-\mu_{1}^{(t+1)}\right)\left(Y_{i}-\mu_{1}^{(t+1)}\right)^{T}}{\sum_{i=1}^{n} T_{j, i}^{(t)}}, j=1,2$
  4. 检查是否收敛.若 $\left\|\Theta^{(t+1)}-\Theta^{(t)}\right\|<\varepsilon$,则停止,否则t=t+1, 转 (2) 。

首先进行一维数据的高斯混合模型参数估计。pdf是一维高斯分布的概率密度函数： $$ f(x)=\frac{1}{\sigma \sqrt{2 \pi}} e^{-\frac{(x-\mu)^{2}}{2 \sigma^{2}}} $$

In [2]:
def pdf(x, mu, var):
    res = 1.0/(np.sqrt(var)*np.sqrt(2*np.pi))*np.exp(-(x-mu)**2/(2*var))
    return res

EM 算法的 E 步骤：估计隐变量 T 和混合参数 P。

In [3]:
def step_E(X, mu, var, P):
    n_points, n_components = len(X), len(P)
    pdfs = np.zeros(((n_points, n_components)))
    for i in range(n_components):
        for j in range(n_points):
            pdfs[j, i] = P[i] * pdf(X[j], mu[i], var[i])
    T = pdfs / pdfs.sum(axis=1).reshape(-1, 1)
    P = T.sum(axis=0) / n_points
    return T, P

EM 算法的 M 步骤：估计每个高斯分布的均值和方差

In [4]:
def step_M(X, T):
    n_components = T.shape[1]
    mu = np.zeros((n_components, 1))
    for i in range(n_components):
        mu[i] = np.average(X, axis=0, weights=T[:, i])
    n_components = T.shape[1]
    var = np.zeros((n_components, 1))
    for i in range(n_components):
        var[i] = np.average((X - mu[i]) ** 2, axis=0, weights=T[:, i])
    return mu, var

EM 算法的总函数。合理地选择初值，计算初始的参数，当迭代之后参数小于阈值 eps 后，停止迭代。

In [5]:
def EM(n_components, X, eps):
    n_points = len(X)
    mu = [0, 3]
    var = [1, 1]
    P = [1 / n_components] * 2
    T = np.ones((n_points, n_components)) / n_components
    P = T.sum(axis=0) / n_points
    theta = np.array([P[0], P[1], mu[0], mu[1], var[0], var[1]])
    old_theta = np.zeros(theta.size)
    while np.linalg.norm(theta - old_theta) > eps:
        T, P = step_E(X, mu, var, P)
        mu, var = step_M(X, T)
        old_theta = theta
        theta = np.array([P[0], P[1], mu[0], mu[1], var[0], var[1]])
        
    return mu, var, P

然后进行二维高斯混合模型的参数估计。代码实现上基本没有变化，只修改了参数的初值和长度。二维高斯分布的概率密度函数使用了 scipy 中的 multivariate_normal.pdf 函数。

In [10]:
def step_E(X, mu, var, P):
    n_points, n_components = len(X), len(P)
    pdfs = np.zeros(((n_points, n_components)))
    for i in range(n_components):
        pdfs[:, i] = P[i] * multivariate_normal.pdf(X, mu[i], np.diag(var[i]))
    T = pdfs / pdfs.sum(axis=1).reshape(-1, 1)
    P = T.sum(axis=0) / n_points
    return T, P

def EM(n_components, X, eps):
    n_points = len(X)
    mu = [[0, 0], [2, 0], [0, 2]]
    var = [[1, 1], [1, 1], [1, 1]]
    P = [1 / n_components] * 3
    T = np.ones((n_points, n_components)) / n_components
    P = T.sum(axis=0) / n_points
    theta = np.array([P[0], P[1], P[2], mu[0][0], mu[0][1], mu[1][0], mu[1][1], var[0][0], var[0][1], var[1][0], var[1][1]])
    old_theta = np.zeros(theta.size)
    while np.linalg.norm(theta - old_theta) > eps:
        T, P = step_E(X, mu, var, P)
        mu, var = step_M(X, T)
        old_theta = theta
        theta = np.array([P[0], P[1], P[2], mu[0][0], mu[0][1], mu[1][0], mu[1][1], var[0][0], var[0][1], var[1][0], var[1][1]])
    return mu, var, P
